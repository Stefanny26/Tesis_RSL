## ANÁLISIS DE RESULTADOS

### Cálculo del SUS Score (Sección 2):
```
1. Convertir respuestas:
   - Preguntas impares (1,3,5,7,9): Restar 1 del valor
   - Preguntas pares (2,4,6,8,10): Restar valor de 5

2. Sumar todos los valores convertidos

3. Multiplicar por 2.5

Resultado: 0-100 puntos
- <50: Inaceptable
- 50-70: Marginal
- 70-85: Bueno
- 85-100: Excelente
```

### NPS (Net Promoter Score) - Pregunta 7.5:
```
Promotores (Definitivamente sí): +100
Pasivos (Probablemente sí): 0
Detractores (resto): -100

NPS = (% Promotores - % Detractores)
```
